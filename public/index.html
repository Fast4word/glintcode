Getting Started Syntax Functions DOM Examples
# Getting Started Include `glint.js`: ```html ````
# Syntax Glint uses JavaScript syntax with extra built-in functions. Example: ```glint let name = "Luke" if(name == "Luke") { print("Hello!") } ```
# Built-in Functions | Function | Description | | ------------------------ | ----------------------- | | print() | Prints to the console | | println() | Prints a line | | warn() | Console warning | | error() | Console error | | random(min,max) | Random integer | | upper(text) | Uppercase text | | lower(text) | Lowercase text | | length(obj) | Length of string/array | | first(array) | First item | | last(array) | Last item | | contains(array,value) | Checks for a value | | create(element,id,class) | Creates an HTML element |
# DOM Creating an element: ```glint create("h1","title","heading") title.textContent = "Hello World" document.body.appendChild(title) ``` Creating a button: ```glint create("button","btn","primary") btn.textContent = "Click Me" btn.onclick = () => { print("Clicked!") } document.body.appendChild(btn) ```
# Example Program ```glint print("Welcome!") create("h1","title","hero") title.textContent = "GlintCode" create("button","button","primary") button.textContent = "Press Me" button.onclick = () => { print(random(1,100)) } document.body.appendChild(title) document.body.appendChild(button) ```
